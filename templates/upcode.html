Hướng dẫn cách up code hoặc dự án lên Github

Cài đặt và sử dụng Git trên Ubuntu

Xin chào các bạn , bài này mình sẽ hướng dẫn về cách up một file chứa code hoặc những dự án về mảng nào đó nằm trong các folder . Mình cũng tình cờ tìm được hướng dẫn và chỉ lại cho các bạn thôi . Cách làm cũng không khó lắm đâu , vài bước là các bạn có thể lưu các file của mình lên và sau tìm lại rất thuận tiện , phòng trường hợp bị xóa mất .

Bài này mình chỉ hướng dẫn trên hệ điều hành Ubuntu thôi nhé !

Đầu tiên các bạn mở Terminal của mình lên ( Có thể dùng phím tắt Ctrl + Alt + T )

Bước đầu tiên ta cài đặt Git trước : 

        $ sudo apt-get install git

Các bạn nhập password của mình để bắt đầu tiến trình cài đặt , sau khi load xong repository , ta nhấn Y rồi Enter để bắt đầu download và cài đặt Git .

Sau khi thấy chạy xong , các bạn kiểm tra xem Git đã được cài chưa , gõ lệnh : 

        $ git --version

Bảng sẽ hiện lên git version 2.7.4 nghĩa là ta đã cài thành công và phiên bản hiện tại của mình là 2.7.4

Tiếp theo bạn truy cập vào Github.com là trang web sẽ lưu các file bạn muốn đẩy lên .

Bạn dùng email của mình để đăng kí và đăng nhập vào Github , bước này đơn giản bạn tự làm nhé .

Oke , để đó ta quay lại với các lệnh trên Terminal . Ta dùng tài khoản các bạn vừa lập để vào Git : 

         $ git config --global user.name  "< Tên tài khoản Github của bạn >"
         $ git cònig --global user.email  "< Email bạn dùng để đăng kí >"

Ví dụ :

Tên tài khoản Github của mình là quangtien18 , email là tientq1801@gmail.com , mình gõ lệnh như sau : 

         $ git config --global user.name "quangtien18"
         $ git config --global user.email "tientq1801@gmail.com"

Tiếp theo ta khởi tạo folder của mình , hiểu đơn giản là tạo một New folder trên máy tính vậy . Folder này là nơi chứa những thứ ta muốn đưa lên Github . 

         $ git init < Tên folder >

Ví dụ : Mình tạo folder có tên là FlaskTutorial - đây là tên hiển thị trên web để vào các file lưu trên đó nên các bạn lấy tên cho phù hợp nhé, gõ như sau : 

         $ git init FlaskTutorial

Các bạn mở nơi chứa files lên sẽ thấy một folder mới được tạo tên bạn vừa dùng , của mình là FlaskTutorial .

Tiếp theo ta di chuyển tới folder FlaskTutorial vừa tạo . 

         $ cd FlaskTutorial

Tới đây , bạn hãy copy những file bạn muốn up lên và paste vào folder FlaskTutorial , hoặc bạn cũng có thể tạo một thư mục mới trong folder bằng cách gõ : 

         $ gedit test.py

Tại đây ta đã mở một chương trình gedit là một trình soạn thảo có sãn trên Ubuntu , nó cũng giống như notepad . File này có tên test.py , trong file này bạn có thể viết bất kì thứ gì , ví dụ một đoạn code về chương trình bạn đang làm hoặc một chương trình Hello World ! hoặc một bài hát là thư tỏ tình :D ,....Vì mình đã có file sẵn nên mình chỉ copy và paste , không tạo mới .

Sau khi copy xong bạn sử dụng câu lệnh : 

         $ git status

Câu lệnh này để kiểm tra trạng thái git hiện tại .

Hình trên những file hiện lên màu đỏ có nghĩa là những file đó chưa được thêm vào chỉ mục ở trên Git . Để thực hiện nó ta làm tiếp như sau : 

         $ git add < Tên file >

Ví dụ : ở đây mình có 4 file , mình thực hiện liên tục là 

         $ git add templates/
         $ git add app.py
         $ git add data.py
         $ git add data.pyc

Sau khi thực hiện xong ta quay lại bước cũ :

         $ git status

Bạn thấy những file của chúng ta đã chuyển sang màu xanh , tức là ta có thể đưa những file này lên Github

Tiếp theo ta phải commit tức là tập hợp các tập tin lại và đánh dấu cho nó một nội dung , cũng để cho người xem có thể biết bạn đang sử dụng những file đó cho vấn đề gì . 

         $ git commit -m  "< Nội dung >"

Ví dụ : Những file này của mình là một hướng dẫn về Flask , mình sẽ lưu với nội dung : Hướng dẫn Flask 

         $ git commit -m "Hướng dẫn Flask"

Oke , ngừng gõ lệnh ta chuyển sang trang wed của Github , ta tiến hành tạo một kho chứa ở trên trang chủ như hình dưới đây

Các bạn tạo New repository . tại Repository name các bạn lưu ý phải tạo tên đúng như tên folder các bạn đã tạo ban đầu , của mình là FlaskTutorial , các bạn có thể để chế độ Public , tức là mọi người có thể thấy file của bạn và có thể tải về hoặc bạn để chế độ Private chỉ mình bạn thấy .

Khi bạn nhấn Create repository sẽ ra như hình dưới

Bây giờ ta sẽ kết nối git ở local tới git trên trang web bằng lệnh : 

         $ git remote add < Tên > < url >

Trong đó 'Tên' là tên bạn đặt bất kì , ủl chính là đường link hiện ở hình trên mà mình đang bôi đen , mình sẽ coppy đường link đó lại và paste vào đây .

Ví dụ : mình lấy 'Tên' là 'MyGit'

         $ git remote add MyGit https://github.com/quangtien18/FlaskTutorial.git

Bước cuối cùng , để đưa file lên ta dùng lệnh :

         $ git push < Tên > < Nhánh >

Ví dụ : 'Tên' chính là tên mình lấy ở trên , của mình là 'MyGit' và 'Nhánh' ta sẽ mặc định là 'master'

         $ git push MyGit master

Sau đó nó sẽ yêu cầu bạn nhập username và password tài khoản bạn đăng kí trên trang Github
